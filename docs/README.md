# Book Builder Documentation

This directory contains documentation for the Book Builder project. We keep documentation separate from the source code to maintain a clean project structure.

## Contents

- [Architecture](./architecture.md): Overview of the technical stack and key concepts.
- [Ideas](./ideas.md): Brainstorming and future feature planning.
- [QA Log](./qa_log.md): A log of bugs and findings during the QA process.

## Goal

The goal of this project is to build a robust territory balancing and assignment tool for sales operations ("RevOps").
